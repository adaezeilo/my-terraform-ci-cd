# my-terraform-ci-cd
# Terraform CI/CD Pipeline on AWS

## Project Overview
This project demonstrates how to automate infrastructure deployment on AWS using Terraform and GitHub Actions.

## What This Project Does
- Uses Terraform to provision AWS infrastructure
- Automates deployment using GitHub Actions
- Implements Infrastructure as Code (IaC)

## Tools Used
- Terraform
- AWS (EC2)
- GitHub Actions

## Workflow
1. Push code to GitHub
2. GitHub Actions runs Terraform
3. Infrastructure is created automatically on AWS

## Outcome
This reduces manual setup and ensures consistent infrastructure deployment.
